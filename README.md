# Computer Vision Lab File

This repository contains Jupyter Notebook files for various computer vision lab experiments, covering fundamental topics such as image filtering, edge detection, segmentation, and more.

## Requirements

- Python 3.x
- Jupyter Notebook
- OpenCV
- NumPy
- matplotlib

Install dependencies:
```bash
pip install -r requirements.txt
```

## How to Use

1. Clone this repository.
2. Open the Jupyter notebooks using `jupyter notebook`.
3. Follow the instructions and run the code cells one by one.

## Contents

- `Lab1_Image_Filtering.ipynb`: Basic image filtering techniques.
- `Lab2_Edge_Detection.ipynb`: Edge detection algorithms.
